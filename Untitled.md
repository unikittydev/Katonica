### Основная информация

```
Основное:
- Название
- Ареал 
- Время существования
- Происхождение
```

```
Экология:
- Общий характер климата
- Особенности ландшафта (наличие рек, гор и т.д.)
- Неблагоприятные факторы
```

### Развитие

```

- История ведения войн
```

```
Технологии:
* Уровень технологий
* Ремесло
```
### Устройство общества

Государственность:
